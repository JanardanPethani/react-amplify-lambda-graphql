## Configure amplify for BE changes

### Add amplify globally

```sh
npm i -g @aws-amplify/cli

# Create and setup user
amplify configure
```

### To add new API

```sh
amplify add api
```

### To add function

```sh
amplify add function
```

### To push changes

```sh
amplify push
```
